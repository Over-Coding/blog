1、前言

因es版本原因，es5.x 以上不能按原来插件的方式安装elasticsearch-head，故需要新的方式安装使用。我这是在win10下安装测试的。

2、环境准备

jdk1.8 以上。

3、安装ElasticSearch

3.1、下载

官网地址：https://www.elastic.co/cn/downloads/elasticsearch
按自己的需求下载相应的版本，我这里选择的是 5.6.16版本的zip包。
提示：es5.x 以上版本最低需要jdk1.8。
